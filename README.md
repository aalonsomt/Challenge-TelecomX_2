# 📊 Telecom X – Análisis y Preparación de Datos para Predicción de Churn

## 📌 Descripción del Proyecto

Este proyecto forma parte del **Challenge Telecom X** y tiene como objetivo analizar los datos de clientes de una empresa de telecomunicaciones para **identificar factores asociados a la cancelación del servicio (Churn)**.

El análisis se centra en la **exploración, limpieza y preparación de los datos**, con el fin de comprender el comportamiento de los clientes y preparar el dataset para futuros modelos de **Machine Learning**.

---

## 🎯 Objetivo del Análisis

El objetivo principal del proyecto es:

* Analizar las variables que influyen en la **cancelación de clientes**.
* Identificar patrones entre los servicios contratados, características del cliente y churn.
* Preparar el dataset para la construcción de modelos predictivos.

Comprender estos factores permite a las empresas **reducir la pérdida de clientes y mejorar estrategias de retención**.

## 📊 Proceso de Análisis

### 1. Importación de Datos

Se cargó el dataset que contiene información sobre clientes, servicios contratados y estado de cancelación.

### 2. Limpieza y Preparación

Se realizaron varias transformaciones en los datos:

* Eliminación de columnas irrelevantes
* Conversión de variables categóricas
* Corrección de nombres de columnas
* Transformación de la variable **Churn** a formato numérico para análisis.

### 3. Análisis Exploratorio de Datos (EDA)

Se exploraron las relaciones entre variables mediante:

* Distribuciones de variables
* Comparación entre clientes que cancelan y los que no
* Matriz de correlación entre variables numéricas

### 4. Visualización

Se generaron gráficos para identificar patrones y posibles variables relevantes para la predicción del churn.
